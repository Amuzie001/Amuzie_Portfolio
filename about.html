About Me

I'm Amuzie Ebeyi, a dedicated Cloud and DevOps Engineer passionate about automating infrastructure, improving deployment workflows, and optimizing system performance across cloud platforms.

I specialize in building secure and scalable cloud environments on Azure and AWS using tools such as Terraform, Ansible, and Python. My expertise also extends to containerization (Docker, Kubernetes), CI/CD pipelines (GitHub Actions, Jenkins, Azure DevOps), and infrastructure monitoring using Grafana and Prometheus.

With a strong foundation in Linux/Windows administration and networking, I focus on delivering robust, cost-efficient, and compliant cloud solutions that align with business goals and security best practices.
